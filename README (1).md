# 🌊 Dreadwaters

> **A First-Person Horror Escape Room built with Unity**

Dreadwaters is a 3D first-person horror escape game where players must explore a flooded office, solve environmental objectives, avoid a relentless monster, and escape before the rising water claims them.

🎮 **Play the game here:** [Dreadwaters on Unity Play](https://play.unity.com/id/games/5def0fdc-7c9b-48d2-bedb-bf5bfc4e1619/dreadwater?utm_source=chatgpt.com)

---

## 📖 Story

What began as an ordinary late-night shift quickly became a nightmare.

Without warning, dark floodwaters began pouring into the office building, trapping everyone inside. As the water continues to rise, a mysterious entity stalks the empty corridors, hunting anyone still alive.

You are the only survivor.

Your only chance of escape is to recover three missing clues hidden throughout the office and unlock the emergency exit before the flood—or the monster—claims you.

---

## 🎯 Objective

Your mission is simple:

* 🔑 Find the **Key**
* 🎂 Find the **Cake**
* 📎 Find the **Stapler**
* 🚪 Unlock the Exit Door
* 🌊 Escape before the floodwaters rise too high
* 👁️ Avoid the monster at all costs

---

## 🕹️ Controls

| Key         | Action                              |
| ----------- | ----------------------------------- |
| **W A S D** | Move                                |
| **Mouse**   | Look Around                         |
| **F**       | Open Doors                          |
| **E**       | Collect Clues                       |
| **R**       | Replay (Game Over / Escape Screen)  |
| **Esc**     | Return to Main Menu (Escape Screen) |

---

## 🗺️ Clue Locations

| Clue       | Location        |
| ---------- | --------------- |
| 🔑 Key     | Break Room      |
| 🎂 Cake    | Kitchen Counter |
| 📎 Stapler | Conference Room |

Collect all three clues to unlock the exit door.

## 👻 Gameplay Features

* 🌊 Rising floodwater that increases the urgency to escape.
* 👁️ AI-controlled roaming monster that chases the player.
* 🔑 Interactive clue collection system.
* 🚪 Functional office doors with player interaction.
* 🎯 Dynamic objective tracking UI.
* 💀 Game Over screen upon being caught by the monster.
* 🏆 Escape ending with replay and main menu options.
* 🎵 Horror-themed UI, atmosphere, and sound effects.
